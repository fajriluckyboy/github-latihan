# Security Policy

## Melaporkan Celah Keamanan

Jika kamu menemukan celah keamanan di repo ini,
mohon laporkan secara privat melalui:

GitHub → Security → Advisories → Report a vulnerability

Jangan laporkan celah keamanan via Issues publik
karena bisa dieksploitasi sebelum diperbaiki.

## Versi yang Didukung

| Versi | Didukung |
|-------|----------|
| v1.0.0 | ✅ |

## Waktu Respons

Laporan akan ditinjau dalam 7 hari kerja.
